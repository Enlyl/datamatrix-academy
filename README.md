# 🌌 DataMatrix Academy: Data Science Journey

An interactive, gamified Data Science course designed to take you from Zero to Junior. Featuring a cyberpunk aesthetic, pedagogical analogies, and practical hands-on examples.

## ✨ Key Features

- **📚 9 Comprehensive Modules**: From Python basics to advanced Machine Learning.
- **🧠 Pedagogical Approach**: Complex concepts explained through relatable analogies (e.g., NumPy arrays as "Egg Cartons").
- **💻 Interactive Code Examples**: Real-world practical snippets for Python, Pandas, SQL, and Scikit-Learn.
- **🎮 Gamified Experience**: Unlock achievements, find hidden secrets (Easter Eggs), and track your progress.
- **🎨 Dynamic Themes**: Switch between **Cyberpunk**, **Retro Pixel**, and **Matrix** styles.
- **📖 Knowledge Hub**: Built-in Glossary and an enriched Cheat Sheet with definitions and use cases.

## 🚀 Curriculum Overview

1. **🐍 Python Basics**: Variables, lists, and core logic.
2. **🔢 NumPy**: High-performance vector computations.
3. **🐼 Pandas**: Data manipulation and "Excel on steroids."
4. **🔍 SQL**: The English language of databases.
5. **📊 Visualization**: Storytelling with Matplotlib and Seaborn.
6. **📈 Statistics**: The mathematical soul of Data Science.
7. **🕵️ EDA**: Exploratory Data Analysis—becoming a data detective.
8. **⚙️ Feature Engineering**: Turning raw data into model fuel.
9. **🤖 Machine Learning**: Predictive modeling (Regression, Classification, Clustering).

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Routing**: React Router 6
- **Styling**: Vanilla CSS (Custom Variable System)
- **Syntax Highlighting**: PrismJS
- **Data**: Static JSON-based lesson architecture

## 🏗️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Enlyl/datamatrix-academy.git
   ```

2. **Install dependencies:**
   ```bash
   cd datamatrix-academy
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
